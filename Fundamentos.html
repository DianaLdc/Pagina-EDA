Fundamentos de programación

¿Qué es programar?

Programar es escribir instrucciones para que una computadora realice una tarea específica. Estas instrucciones deben seguir una sintaxis determinada, como cualquier lenguaje.

Ejemplo de programación: el coche

Una computadora en un coche puede entender comandos como:


Start;
Go North 1 Blocks;
Go East 2 Blocks;
Go North 5 Blocks;
Go East 2 Blocks;
Stop;

Este es un ejemplo de sintaxis:


<instrucción> ::= Start | Stop | Go <dirección> <número> Blocks
<dirección> ::= North | East | South | West
<número> ::= 1 | 2 | 3 | 4 | 5

Algoritmo vs. Programa

Un algoritmo es la secuencia de pasos para resolver un problema. Un programa es la implementación del algoritmo en un lenguaje de programación.

Errores comunes

Errores de ejecución ocurren cuando una instrucción no se puede realizar


Start;
Go North 1 Blocks;
Go East 3 Blocks;

Errores de lógica ocurren cuando el programa no llega al resultado deseado


Start;
Go North 1 Blocks;
Go East 2 Blocks;
Go North 5 Blocks;
Go West 2 Blocks;
Stop;

Operadores aritméticos

  • + Suma
  • - Resta
  • * Multiplicación
  • / División

int a = 10;
int b = 5;
int c = 2;
int resultado = a + b * c;
System.out.println("Resultado: " + resultado); // 20

Variables y tipos

Las variables almacenan datos que pueden cambiar. Deben declararse indicando su tipo:


int edad = 19;
double pi = 3.1416;

Expresiones y precedencia

La prioridad de operadores afecta el resultado:


int resultado = (a + b) * c; // Paréntesis primero

Abreviaciones aritméticas


a += 12; // a = a + 12;
a *= 3;  // a = a * 3;
a -= 5;  // a = a - 5;

Expresiones lógicas

Comparaciones devuelven valores booleanos (true o false):


a < b   // menor
a >= b  // mayor o igual
a == b  // igual
a != b  // distinto

Condicionales: if / else


if (condición) {
  // código si es verdadero
} else {
  // código si es falso
}

Repeticiones: while


while (condición) {
  // repetir mientras se cumpla
}
