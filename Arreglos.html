Estructuras de Datos

Arreglos Unidimensionales

Un arreglo unidimensional es una colección de elementos del mismo tipo, almacenados en posiciones contiguas de memoria.

  • Los índices comienzan desde 0.
  • Se accede a los elementos por su posición.

int[] numeros = new int[5];
numeros[0] = 10;
numeros[1] = 20;
System.out.println(numeros[1]); // 20

Arreglos Bidimensionales

Es una matriz de filas y columnas. Se usa para representar tablas o matrices.

  • Se accede usando dos índices: fila y columna.
  • Útil para álgebra matricial, gráficos, etc.

int[][] matriz = {
  {1, 2, 3},
  {4, 5, 6}
};
System.out.println(matriz[1][2]); // 6

ArrayList

ArrayList es una clase de Java que implementa una lista dinámica de objetos.

  • No requiere tamaño fijo.
  • Permite agregar y eliminar elementos dinámicamente.

ArrayList<String> lista = new ArrayList<>();
lista.add("A");
lista.add("B");
System.out.println(lista.get(1)); // B

Listas Enlazadas

Son estructuras donde cada nodo contiene un valor y una referencia al siguiente nodo. Se pueden clasificar en:

  • Simple: cada nodo apunta al siguiente.
  • Doble: apuntan al anterior y al siguiente.
  • Circular: el último apunta al primero.

class Nodo {
  int dato;
  Nodo siguiente;

  Nodo(int d) {
    dato = d;
    siguiente = null;
  }
}

Pilas

Una pila funciona bajo el principio LIFO (Last In, First Out). Solo se puede acceder al elemento superior.

  • Push: agregar al tope.
  • Pop: quitar del tope.
  • Peek: ver el tope sin eliminarlo.

Stack<Integer> pila = new Stack<>();
pila.push(10);
pila.push(20);
System.out.println(pila.pop()); // 20

Colas

Una cola funciona bajo el principio FIFO (First In, First Out). Se agrega por detrás y se elimina por delante.

  • Enqueue: agregar al final.
  • Dequeue: quitar del frente.
  • Front: mirar el primer elemento.

Queue<String> cola = new LinkedList<>();
cola.add("A");
cola.add("B");
System.out.println(cola.remove()); // A
